## ⚡ About Me

> Full-Stack developer who builds complete products — from pixel-perfect UI to production-grade backend systems and real-world deployments.

I'm **S M Samiul Hasan**, a Full-Stack Web & App Developer from Rangpur, Bangladesh. I don't just write code — I architect and ship complete digital products. My journey started at **Inspired IT Rangpur**, leveled up through **Programming Hero (Batch 12)**, and is currently pushing forward with the **Next Level AI-Driven Software Engineering Bootcamp**.

I've built and deployed real platforms including **EZBid.cloud** (a B2B e-procurement system) and **Jamibd** (an AI-powered e-commerce platform), both running in production with real users. I manage my own Linux VPS, configure Cloudflare, handle Docker deployments, and patch kernel vulnerabilities when needed.

**What I actually do:**
- 🌐 Build full-stack web apps with Next.js, Node.js, MongoDB, PostgreSQL
- 📱 Build cross-platform mobile apps with Flutter & Dart
- 🤖 Automate data pipelines with Python, Playwright & Selenium
- ☁️ Deploy and maintain apps on VPS, Vercel, Cloudflare, Docker
- 🛠️ Debug, fix, and maintain existing products end-to-end

> 💡 **Note on "Problem Solving":** My expertise is real-world development — debugging architectural errors, building features, and delivering production-ready software. I'm not a competitive programmer. I solve your business problems with code.
